Habilidades envolvidas

Programação

Design centrado no usuário

Detalhes

código se baseou sobretudo em apredizados de programação tais como: dicionários,listas, condições, variáveis, funções e classes. Ademais, o projeto também proporciou o aprendizado da utilização da plataforma github para desenvolvimento de codigo de maneira conjunta, exigindo cooperação ecomunicação assertiva entre eu e minha dupla além de organização significativa frente a quantidade de linhas presentes no código.
Durante o desenvolvimento foi necessário principalmente pensar e se colocar no lugar do usuário, com base nisso apresentei protótipos do meu jogo para pessoas fora do convívio da faculdade, objetivando testar e otimizar o código que estava construindo.

Caso exista interesse em jogar o protótipo final do inspermoon acesse o link ebaixe os códigos: GitHubInspermonn
